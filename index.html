How to secure an SSH server

edit /etc/ssh/sshd_config and apply the folliwing changes:

commented option value
noPermitRootLoginno
noPubkeyAuthenticationyes
noPasswordAuthenticationno
noUsePAMno

Then restart the sshd service, making sure pub key is in ~/.ssh/authorized_keys
